# Sybink Systems

Sybink Systems was a London-based technology solutions provider offering software development, web and graphic design, and consultancy services. We integrated robust hardware and software to deliver fully tested, end-to-end solutions.

The company ceased operations in 2014. This repository includes software and tools developed as part of the venture, which will be added as time permits.

See [Sybink Systems](https://sybink.com/)
